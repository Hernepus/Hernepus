About randEvent

randEvent is a powerful Lua function library for Garry's Mod that provides an advanced event system for mod developers. Create dynamic, randomized in-game events with ease!

Documentation

Complete documentation is available in the download package. Key commands include:

  • rand help - Display available commands
  • rand load {name} - Load third-party mods
  • rand record {filename} - Record custom events
  • rand connect {server} - Connect to randEvent servers
For advanced usage, please refer to the developer documentation included with the library.
